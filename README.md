# Cookie-tool

### A lightweight extension for developers and testers to view and export cookies for the current website.

## Features:

### List all cookies with HttpOnly, Secure, and SameSite flags.

### Select/deselect cookies, copy to clipboard, or export as .txt.

### Select all / none functionality.

### Clean, dark developer-friendly UI.

### Fully local; no data is sent anywhere.

## Privacy:

### Only accesses cookies when you interact.

### For development and personal use only.
